# Custom Payload Encoder & Obfuscation Framework

## Overview

This project demonstrates how payload encoding and obfuscation techniques can affect signature-based detection systems.

The framework allows users to encode and obfuscate payloads using different techniques and test whether they bypass simple detection logic.

## Features

- Base64 Encoding
- XOR Encoding
- ROT13 Cipher
- Random Character Obfuscation
- String Splitting & Concatenation
- Escape Sequence Encoding
- Signature Detection Simulation

## Technologies Used

- Python 3
- Base64 Library
- Random Library

## Project Structure

Custom-Payload-Encoder

main.py – main program  
encoder.py – encoding functions  
obfuscator.py – obfuscation techniques  
evasion_test.py – detection simulation  
sample_payloads.txt – testing payloads
